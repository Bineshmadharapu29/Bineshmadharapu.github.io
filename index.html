About Us

Dedicated cybersecurity professionals with hands-on experience in securing enterprise environments, penetration testing, and vulnerability assessment.

Mohd Abdul Jameel is a Security Engineer with expertise in vulnerability assessments, network security, and enterprise security. He creates custom professional security assessment reports covering end-to-end project workflows, including threat modeling, SAST, SCA, DAST, and API security testing. In cloud security, he meticulously reviews all GCP services, identifying and addressing misconfigurations. He is passionate about API security and cloud technologies, leveraging tools like Sophos XDR and SonicWall.

He enjoys participating in InfoSec events like Microsoft BlueHat, VulnCon, and BSides, and is committed to staying on the cutting edge of the cybersecurity landscape.

Both are passionate about giving back to the community, mentoring aspiring cybersecurity professionals, and actively engaging in cybersecurity events and knowledge-sharing initiatives.
